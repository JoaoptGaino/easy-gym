The project was developed with these technologies:

# ⚙ Installing

```bash
     #Clone repo
    $ git clone https://github.com/JoaoptGaino/easy-gym

    #Enter repo directory
    $ cd easy-gym

    #Install the dependencies
    $ yarn install

    #Init project
    $ yarn start

```
